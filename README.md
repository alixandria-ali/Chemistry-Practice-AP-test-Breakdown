## Test Overview

- **Format**: 7 Free-Response Questions
- **Time**: 1 hour 45 minutes
- **Topics**: All AP Chemistry units with focus on thermodynamics, kinetics, equilibrium, acids/bases, electrochemistry

## Questions Covered

### Long Free-Response (Questions 1-3)
Each worth 10 points, ~23 minutes each

1. **Ethanol Combustion** - Thermochemistry, stoichiometry, oxidation-reduction, IMFs, equilibrium
2. **Hydrogen Peroxide Decomposition** - Particulate representations, energy diagrams, catalysis, kinetics
3. **Sulfur Trioxide** - Lewis structures, VSEPR, thermodynamics, equilibrium expressions

### Short Free-Response (Questions 4-7)
Each worth 4 points, ~9 minutes each

4. **Hypochlorous Acid/Water** - Acid-base equilibria, titration curves, buffers
5. **NADH Electrochemistry** - Electrochemical cells, Gibbs free energy, redox reactions
6. **Paraffin Wax Phase Changes** - Thermal energy, heat of fusion, particle-level explanations
7. **Phenol Red Spectrophotometry** - pH calculations, Beer's Law, dilution, spectroscopy
