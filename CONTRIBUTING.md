# Contributing to This Project

Thank you for your interest in improving the AP Chemistry Practice Test Breakdown!

## How to Contribute

### 1. Report Errors
If you find any mistakes in:
- Chemical equations
- Calculations
- Explanations
- Formatting

Please create an issue with:
- Question number
- Description of the error
- Suggested correction

### 2. Add Missing Questions
We currently have 7 complete question breakdowns. To add more:

**Required Format for Each Question:**

```markdown

**Topic**: [Specific Topic with AP Code]

[Full question text with all options]
```

### 3. Improve Existing Questions
Enhancements welcome:
- Clearer explanations
- Better diagrams (if you can add images)
- Additional practice problems
- More detailed "Key Concepts" sections

### 4. Add Study Resources
- Formula sheets
- Quick reference guides
- Memory aids and mnemonics
- Links to helpful videos or resources

## Style Guidelines

### Writing Style:
- Clear and concise
- Student-friendly language
- Step-by-step approach
- Include "why" not just "how"

### Formatting:
- Use markdown headers (#, ##, ###)
- Bold important terms with **asterisks**
- Use tables for comparing concepts
- Use code blocks for equations when helpful
- Break long sections into digestible chunks

### Calculations:
- Show all steps
- Include units
- Round appropriately
- Explain what each variable represents

## Review Process

1. **Accuracy**: All chemistry must be correct
2. **Clarity**: Explanations must be understandable
3. **Completeness**: All sections must be filled
4. **Consistency**: Follow existing format
5. **Educational Value**: Should help students learn
